# homebrew-tap

Homebrew tap for [idp](https://github.com/SrimanMayandi02/idp).

## Install

```bash
brew install SrimanMayandi02/tap/idp
```

## What is this repo?

This repo only contains auto-generated Homebrew formulas — pointers to release
binaries built by [GoReleaser](https://goreleaser.com) over in the
[idp](https://github.com/SrimanMayandi02/idp) repo. It is updated automatically
on every release; do not edit `Formula/*.rb` files by hand.
